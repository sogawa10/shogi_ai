# 将棋AI

## 概要

このプロジェクトは，将棋AIを自前実装することを目的とした学習用プロジェクトです．**将棋のルール・状態管理機能**，**ai機能**，そして**GUI機能**を段階的に実装します．既存の将棋エンジンを使用することなく，基礎構造から実装している点が特徴です．

将棋のルールとして，**二歩の禁止**，**行き所のない駒の禁止**，**打ち歩詰めの禁止**，**千日手判定**，**入玉宣言法**を実装しました．

## 実行環境

このプロジェクトは，Pythonで実装されています．動作確認は主にWindows環境で行っていますが，標準的なPython環境であれば他OSでも動作する可能性が高いです．

## インストール方法

リポジトリをクローンした後，必要に応じて仮想環境を作成し，依存ライブラリをインストールしてください．現時点では標準ライブラリのみで動作する構成を想定していますが，将来的に外部ライブラリを導入する可能性があります．

## 使い方

1. python -m shogi_ai
2. 上記の対局スクリプトを直接実行することで，人間同士の対戦や，人間対AIの対戦を行うことができます．
3. 駒を動かしたい場合は，**「5 4 5 3」** のように，元居る場所の x座標，y座標，進みたい場所の x座標，y座標 をスペースを含めて入力してください．成りたい場合は，**「5 4 5 3 成」** のように，最後に「成」と追加してください．
4. 駒を打ちたい場合は，**「歩 5 5」** のように，打ちたい駒の名前と，進みたい場所の x座標，y座標 をスペースを含めて入力してください．
